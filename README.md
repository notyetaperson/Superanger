\# Pixel Rearrangement Tool



A beautiful, high-performance web application that rearranges pixels from a \*\*source image\*\* to match the colors and structure of a \*\*modulator image\*\*. Creates stunning mosaic-like effects through intelligent pixel matching.



!\[Pixel Rearrangement Tool](https://via.placeholder.com/800x400/667eea/ffffff?text=Pixel+Rearrangement+Tool)



\## ✨ Features



\- \*\*Smart Pixel Matching\*\*: Uses optimized algorithms (spatial hashing + LAB color space) for high-quality results

\- \*\*Real-time Processing\*\*: Chunked processing with smooth progress updates

\- \*\*Beautiful Animations\*\*: Flying pixel particles that visualize the rearrangement

\- \*\*Modern UI\*\*: Glassmorphism design with smooth animations and custom cursor

\- \*\*Performance Optimized\*\*: 

&#x20; - Automatic quality reduction for large images

&#x20; - Spatial hashing for fast nearest-neighbor search

&#x20; - LAB color space for perceptually accurate matching

&#x20; - RequestAnimationFrame-based processing

\- \*\*Browser Compatible\*\*: Works with modern browsers supporting Canvas and ES6 modules
